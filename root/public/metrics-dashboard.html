Debugging Playbook

30-second debugging workflow:

Step 1: Pipeline Health Find failing stage: guard→extract→route→gather→blend→verify. Success rate = completion, verify rate = quality.
Step 2: Confidence Analysis High-confidence failures = model drift. Low-confidence successes = undertrained intents.
Step 3: Search Quality Complex query rate vs result quality shows if search handles difficult queries properly.
Step 4: External Dependencies Slow APIs by location/domain help identify regional issues or provider problems.
