Production Debugging Guide

How to debug production issues in 30 seconds:

Pipeline Health Check which stage is failing: guard→extract→route→gather→blend→verify. Low success rates indicate bottlenecks. Verify rates show result quality.
Confidence Correlation High-confidence failures = model drift. Low-confidence successes = undertrained intents.
External API Context Slow APIs by location/domain help identify regional issues or specific content problems.
Search Quality Complex query rate vs result quality shows if search is handling difficult queries properly.
