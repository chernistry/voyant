Voyant Travel Assistant

Builds trust, then answers fast.

A minimalist, fact-grounded travel assistant that employs LLM-driven NLP for intent classification and slot filling, pulling verified data from trusted APIs with multi-layered hallucination prevention.

Key Features

shield

Anti‑Hallucination

Self‑evaluates to prevent hallucinations, requiring citations and secondary verification.

travel_explore

Web Search

Delivers standard and deep research modes with intelligent page crawling for complex queries, ensuring robust fallback when APIs fail.

psychology

Smart Routing LIVE

Transformers-powered multilingual NLP for smarter routing using zero-shot classification. Reduces LLM calls by 20–40% with a resilient cascade (Regex > Transformers > LLM) and handles multi-turn conversations naturally.

public

Real-Time Data

Connects to Open‑Meteo, REST Countries, and OpenTripMap for verified travel info, with robust retry/backoff to handle external API instability.

search

Enterprise-grade RAG LIVE

Advanced retrieval-augmented generation with Vectara integration for searching and retrieving information from internal knowledge bases, policy documents, and enterprise data sources.

bolt

Fast & Clean

Interactive CLI and REST API with TS validation, structured logging, and comprehensive E2E tests using real agent interactions with LLM-based eval.

Quick Start

# Clone repository and enter root directory
git clone https://github.com/chernistry/voyant/ && cd root

# Install dependencies
npm install

# Start CLI
npm run cli

# Or start HTTP server
npm run dev

# Run tests
npm test

smart_toy

The standard cancellation window for Marriott hotels depends on the rate you booked.

  • Standard Rates: You can cancel up to 24 hours before arrival for no fee. Cancellations within 24 hours of arrival incur a one-night penalty. No-shows are charged the full stay.
  • Advance Purchase Rates: These are generally non-refundable with no changes or cancellations allowed, except for extraordinary circumstances.
  • Flexible Rates: You can cancel up to 6 PM on the day of arrival for a full refund.
  • Group Bookings (10+ rooms): A 72-hour cancellation notice is required, and attrition fees may apply.

Marriott Bonvoy Elite members have enhanced flexibility, with Platinum and above having a 4 PM cancellation window on the day of arrival. Gold members have the standard 24-hour policy.

The documents do not specify a standard cancellation window for Hilton hotels.

Notes on Prompt Engineering

  • Format Priming: Enforced JSON/bullet-point schemas ensure structured, parseable outputs while preventing hallucinations.
  • Few-Shot Prompting: Curated examples in router.md and cot.md train accurate intent classification and slot extraction.
  • Chain-of-Thought: Structured reasoning in cot.md and verify.md decomposes analysis into verifiable steps.
  • Guardrails: blend.md requires fact-grounding + verify.md provides secondary validation of claims.
  • Query Optimization: Rewrites user search queries to reduce noise.

Other Perks

  • Anti-hallucination receipts with self-check and sources.
  • Resilient data blend: APIs → search fallback when providers fail.
  • Smart routing + memory for natural multi-turn conversations.
  • Optional /metrics endpoint (Prometheus or JSON) and structured logs (PII‑redacted by default).
  • Security-conscious: host allowlist, timeouts, retries with jitter, and circuit-like backoff.
  • E2E transcripts and golden tests for the core scenarios.
